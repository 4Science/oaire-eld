# Source Code
The source code related to the project can be found in the 4Science GitHub repositories, in a branch named oaire-eld: 
* REST Contract: https://github.com/4Science/Rest7Contract/tree/oaire-eld
* DSpace REST Backend: https://github.com/4Science/DSpace/tree/oaire-eld
* DSpace Angular Frontend: https://github.com/4Science/dspace-angular/tree/oaire-eld

If admitted to the phase 3 we will produce PRs and patch files to be applied to the official DSpace 7 code base. Moreover, the features are intended to be merged in the DSpace-CRIS 7 code base once the project is completed.
